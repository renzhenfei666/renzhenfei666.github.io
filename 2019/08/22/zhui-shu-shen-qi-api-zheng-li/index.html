所有接口都是GET请求

1. 获取所有分类

类型 值
接口地址 /cats/lv2/statistics
参数 null
实例接口地址 api.zhuishushenqi.com/cats/lv2/st…

2. 获取排行榜类型

类型 值
接口地址 /ranking/gender
参数 null
实例接口地址 api.zhuishushenqi.com/ranking/gen…

3. 获取排行榜小说

类型 值
接口地址 /ranking/:rankId
参数 排行榜的ID,不同字段对应不同的榜单,具体ID从接口2中获取
_id: 周榜ID
monthRank: 月榜ID
totalRank: 总榜ID
实例接口地址 api.zhuishushenqi.com/ranking/54d…

4. 获取分类下小类别

类型 值
接口地址 /cats/lv2
参数 null
实例接口地址 api.zhuishushenqi.com/cats/lv2

5. 根据分类获取小说列表

类型 值
接口地址 /book/by-categories
参数 说明无
gender: 男生:mael 女生:female 出版:press
type: 热门:hot 新书:new 好评:repulation 完结: over 包月: month
major: 大类别 从接口1获取
minor: 小类别 从接口4获取 (非必填)
start: 分页开始页
limit: 分页条数
实例接口地址 api.zhuishushenqi.com/book/by-cat…

6. 获取小说信息

类型 值
接口地址 /book/:booId
参数 booId具体小说的ID
实例接口地址 api.zhuishushenqi.com/book/548d9c…

7. 获取小说正版源

类型 值
接口地址 /btoc
参数 view: 暂时只知道summary这个参数 book: 对应的bookId
实例接口地址 api.zhuishushenqi.com/btoc?view=s…

8. 获取小说正版源于盗版源(混合)

类型 值

参数 view: 暂时只知道summary这个参数book: 对应的bookId
实例接口地址 api.zhuishushenqi.com/atoc?view=s…

9. 获取小说章节(根据小说id)

类型 值
接口地址 /mix-atoc/:bookId
参数 bookId:对应小说id view:暂时只知道chapters
实例接口地址 api.zhuishushenqi.com/mix-atoc/50…

10. 获取小说章节(根据小说源id)

类型 值
接口地址 /atoc 或者/btoc
参数 sourceId:对应小说源id view:暂时只知道`chapters
